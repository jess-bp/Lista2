# Lista2
Exercícios para exercitar a lógica de programação em JavaScript, utilizando somente os conteúdos envolvidos no Módulo 1: Introdução à Lógica de Programação, da Formação Dev Full Stack Junior da +prati e Codifica.

Configurações e Execução:
Para executar os exercícios, você precisa de um ambiente com o Node.js instalado.

Configuração:
Faça o clone deste repositório na máquina local:

# HTTPS:
git clone //github.com/jess-bp/Lista2

# SSH:
git clone git@github.com:jess-bp/Lista2.git
